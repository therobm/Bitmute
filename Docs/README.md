# Bitmute User Guide

Bitmute is a raster image editor for game-development texture work and general-purpose pixel editing, built with .NET MAUI and SkiaSharp. A compact, familiar workspace, layered non-destructive editing, 8/16/32-bit color, and canvases up to 8K+.

![The Bitmute workspace](Images/workspace-overview.png)

# Keyboard Shortcuts

## Tools

Single key selects the tool; **Shift+key** cycles within a flyout group.

| Key | Tool |
|---|---|
| M | Box Select (⇧M: Circle Select) |
| L | Freehand Lasso (⇧L: Poly, Magnetic) |
| W | Magic Wand |
| V | Move |
| C | Crop |
| I | Eyedropper (⇧I: Ruler) |
| B | Brush (⇧B: Pencil, Color Replacement) |
| S | Clone (⇧S: Heal) |
| E | Eraser |
| G | Fill (⇧G: Gradient) |
| U | Line (⇧U: Rectangle, Rounded Rect, Ellipse, Polygon) |
| R | Blur (⇧R: Sharpen, Smudge) |
| O | Dodge (⇧O: Burn, Sponge) |
| T | Text |
| Z | Zoom |
| H | Move Canvas (Hand) |
| P | Pen |
| A | Direct Selection |

## File

| Shortcut | Action |
|---|---|
| `Ctrl+N` | New document |
| `Ctrl+O` | Open |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+Alt+Shift+S` | Export As |

## Edit

| Shortcut | Action |
|---|---|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Undo step |
| `Ctrl+Alt+Z` | Redo step |
| `Ctrl+X` / `Ctrl+C` / `Ctrl+V` | Cut / Copy / Paste |
| `Ctrl+Shift+C` | Copy Merged |
| `Ctrl+Shift+V` | Paste Into |
| `Ctrl+T` | Free Transform |

## Selection

| Shortcut | Action |
|---|---|
| `Ctrl+A` | Select All |
| `Ctrl+D` | Deselect |
| `Ctrl+Shift+I` | Invert Selection |
| `Delete` | Clear selected pixels |
| `Alt+Delete` | Fill with foreground |
| `Ctrl+Delete` | Fill with background |
| `Shift` / `Alt` / `Shift+Alt` (while selecting) | Add / Subtract / Intersect |

## Layer

| Shortcut | Action |
|---|---|
| `Ctrl+J` | Duplicate Layer |
| `Ctrl+E` | Merge Down |
| `Ctrl+Shift+E` | Merge Visible |

## Image & Filters

| Shortcut | Action |
|---|---|
| `Ctrl+Alt+I` | Image Size |
| `Ctrl+I` | Invert Colors |
| `Ctrl+F` | Repeat Last Filter |

## View & Zoom

| Shortcut | Action |
|---|---|
| `Ctrl++` / `Ctrl+-` | Zoom in / out |
| `Ctrl+0` | Fit on Screen |
| `Ctrl+R` | Toggle Rulers |
| `Alt+Scroll` | Zoom |
| `Scroll` / `Ctrl+Scroll` | Pan vertically / horizontally |
| `Space`+drag | Pan |

## Color

| Shortcut | Action |
|---|---|
| `X` | Swap foreground/background |
| `D` | Reset to black/white |

## Committing

| Shortcut | Action |
|---|---|
| `Enter` | Commit the current text/transform/tool operation |
| `Esc` | Cancel the current operation |
